

# Target Brazil Analytics

## Project Overview
This project analyzes Target Brazil's e-commerce operations using a modern SQL analytics workflow.

## Objectives
- Build a layered data warehouse (Bronze, Silver, Gold)
- Perform data cleaning and transformation
- Create analytical SQL queries
- Generate business insights and KPIs
- Build visualizations and reports

## Project Structure
datasets/        Raw and processed datasets
docs/            Project documentation
sql/             SQL scripts and transformations
visualizations/  Charts and dashboards

## Tech Stack
- SQL
- PostgreSQL
- GitHub
- Power BI / Tableau
